Du=max⁡1≤i≤tu{Dvi+edgeu,vi}D_u = \max_{1 \leq i \leq t_u}\{D_{v_i} + edge_{u, v_i}\} Du​=1≤i≤tu​max​{Dvi​​+edgeu,vi​​}

Fu=max⁡1≤j<i≤tu{Dvi+edgeu,vi+Dvj+edgeu,vj}F_u = \max_{1 \leq j < i \leq t_u}\{D_{v_i} + edge_{u, v_i} + D_{v_j} + edge_{u, v_j}\} Fu​=1≤j<i≤tu​max​{Dvi​​+edgeu,vi​​+Dvj​​+edgeu,vj​​}

2 最近公共祖先

2.1 原理

基本思路: 先将两个节点跳到同一层，再同时向上跳只到找到共同祖先

2.2 实现

2.2.1 倍增法

最常见的一种求LCA的算法，设计数组 Fu,kF_{u, k}Fu,k​ 表示节点 uuu 向上(根节点)跳 2k2^k2k 次到达的节点，每次跳跃按照 kkk 从大到小的顺序尝试

时间复杂度 O(qlogn)O(qlogn)O(qlogn)

2.2.2 树链剖分/重链剖分

代码长度较短的一种做法，先对树进行重链剖分，再沿着重链不断往上跳

时间复杂度 O(qlogn)O(qlogn)O(qlogn)

2.2.3 Tarjan算法

  • 这是一个 离线算法 使用时要注意

  1. 考虑离线操作，先将询问排序

  2. 自根节点开始DFS遍历整个树

  3. 在进入一个节点时，先将他在并查集中的父亲设置为自己，在遍历完所有子节点后，处理与该节点相关的所有询问，如果询问中另一个节点为已经访问过的节点，那么该节点所在的并查集的代表元即为他们的最近公共祖先

  4. 回溯离开节点时，在并查集上将它所处的集合与其父节点所处的集合合并

时间复杂度 O(n+m)O(n + m)O(n+m)

2.3 应用

2.3.1 树上差分

问题描述

对于 nnn 个节点的树，进行 kkk 次修改操作，每次将 sss 到 ttt 的路径上的点权增加 xxx，查询每个点的最终值.

思路

对于每一次修改的路径 $ s \to t $ 我们可以把将其拆分为 $ s \to LCA(s, t) \to t$ 两段，考虑到这是一个区间修改单点查询的问题，我们可以用差分思想将修改操作进行转换:

建立一个差分数组 dud_udu​ 对应根节点到 uuu 的节点路径的差分值，对于每一个修改，我们将 $ s \to LCA(s, t) $ 和 $ LCA(s, t) \to t$ 两条路径都加上 xxx，由于节点 LCA(s,t)LCA(s, t)LCA(s,t) 被重复计算了一次，所以该点的点权还要减去 xxx。因此，只需将 dsd_sds​ 和 dtd_tdt​ 增加 xxx，dLCA(s,t)d_{LCA(s, t)}dLCA(s,t)​ 和 dfaLCA(s,t)d_{fa_{LCA(s, t)}}dfaLCA(s,t)​​ 减少 xxx 即可，使用倍增求 LCA。

最后遍历整棵树，还原出每个点的点权并统计答案。
