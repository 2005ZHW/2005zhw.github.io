Her favorite algorithm thus far is “bubble sort”. Here is Bessie’s initial implementation, in cow-code, for sorting an array AAA of length NNN.

After testing her code on several arrays, Bessie learns an interesting observation: while large elements can be pulled to the end of the array very quickly, it can take small elements a very long time to “bubble” to the front of the array (she suspects this is how the algorithm gets its name). 题目大意

求对序列 ana_nan​ 双向冒泡排序的 循环 次数 ( 1≤N≤1051 \leq N \leq 10^51≤N≤105 ).

题解

通过模拟题目所给的程序，我们可以注意到每次循环其实就是将未排序的部分中最小值和最大值分别移到左右端.

对于任意位置的xxx，就是将在其前面大于xxx的数与在其后面小于xxx的数交换，故答案为 maxi=1n{∑j=1i[aj>ai]}max_{i=1}^{n}\{\sum_{j=1}^{i}[a_j>a_i]\}maxi=1n​{∑j=1i​[aj​>ai​]} .

我们将序列ana_nan​进行离散化，可以得到答案是 maxi=1n{∑j=1i[aj>i]}max_{i=1}^{n}\{\sum_{j=1}^{i}[a_j>i]\}maxi=1n​{∑j=1i​[aj​>i]} ，但是这样的时间复杂度是 O(n2)O(n^2)O(n2) 并不能通过此题，考虑使用树状数组优化查询第二维，可将时间复杂度降为 O(nlogn)O(nlogn)O(nlogn) .

注意: 原本有序的序列也会进行一次循环，答案为 111 .
